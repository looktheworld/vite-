# 搭建vite工程

vite官网搭建工程指南: https://vitejs.dev/guide/#scaffolding-your-first-vite-project

- 使用自选配置去构建vite项目: 
  ```
  yarn create vite
  ```

- 构建一些具备框架功能的预设模板(不需要你自选配置)
  ```
  yarn create vite my-react-ts-app --template react-ts # 还有更多配置可以参阅官网
  ```
